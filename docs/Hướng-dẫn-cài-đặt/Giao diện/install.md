#### 2. Lấy mã git SSH và import vào trên Appsmith:

1. Fork repository.
2. Lấy mã SSH và import vào trên Appsmith
3. Tạo một nhánh mới cho tính năng hoặc bản sửa lỗi của bạn.
3. Commit thay đổi và đẩy chúng lên nhánh của bạn.
